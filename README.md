**Основы Python. Тема 2. Переменные и типы данных языка Python и работа с ними**


**Краткий конспект**


Процесс создания переменной и присваивания ей значения называется инициализацией переменной. a = 'name'


- Название переменной может содержать только буквы, числа и нижнее подчеркивание;
- Название не может начинаться с числа;
- Название должно быть осмысленным и понятным (например, my_age или user_name);
- Изменяемые в программе переменные всегда называются буквами в нижнем регистре.


Константы пишутся в верхнем регистре. CONST1 = "Python"

- Строка - str
- Целое число - int

- '+' — сложение
- '-' — вычитание
- '*' — умножение
- / — деление (всегда возвращает float)

Строки обозначаются кавычками. Строки являются итерируемым объектом. Они позволяют обратиться к элементам объекта по индексу. Индексация всех объектов в Python начинается с 0, поэтому обращение к первому символу будет проходить по индексу 0. Само обращение задается с помощью квадратных скобок. Обратная индексация идет от конца строки, где последний элемент будет иметь индекс -1. 


Срез - это подмножество элементов итерируемого объекта, задается следующим образом: mystr[START:STOP:STEP] - берёт срез от номера START, до STOP (не включая его), с шагом STEP.


Самый современный вариант форматирования – это f-строки, которые позволяют писать переменные в фигурных скобках по мере их использования в тексте. 


**Методы строк**
- upper() — преобразует все символы строки в заглавные
- lower() — преобразует все символы строки в строчные
- find(sub) — возвращает индекс первого вхождения подстроки, -1 если не найдено
- index(sub) — возвращает индекс первого вхождения, вызывает ошибку если не найдено
- count(sub) — возвращает количество вхождений подстроки
- isalpha() — возвращает True, если все символы буквы
- isdigit() — возвращает True, если все символы цифры
- strip(chars) — удаляет указанные символы с начала и конца строки (по умолчанию пробелы)
- split(sep, maxsplit) — разбивает строку на список по разделителю
- replace(old, new, count) — заменяет вхождения old на new (count раз, если указано)


**Список** – это последовательность элементов, объединенная единым адресом. lst1 = [1,3.14, "c", "st", [1,3]]


Также как в строке все элементы списка имеют четкую индексацию и могут быть извлечены по индексу в квадратных скобках. Однако, в отличии от строки список может содержать элементы любого типа данных. Список задается с помощью квадратных скобок. Списку доступны все возможности итерируемых объектов из тех, что мы разобрали: срезы, прямая и обратная индексация. 
- len() – функция, которая позволяет определить размер списка;
- sum() – функция, которая позволяет суммировать значения элементов списка;
- map() – функция, которая позволяет применить к каждому элементу списка другую функцию.
- append(x) — добавляет элемент x в конец списка
- insert(i, x) — вставляет элемент x на позицию i (сдвигая остальные вправо)
- extend(iterable) — добавляет все элементы из итерируемого объекта в конец списка
- remove(x) — удаляет первое вхождение элемента x (вызывает ошибку, если элемент не найден)
- pop(i) — удаляет и возвращает элемент на позиции i (по умолчанию последний)
- clear() — удаляет все элементы из списка (очищает список)


Очень похожий на список тип данных – кортеж, по сути является неизменяемым списком. Это значит, что его нельзя редактировать после создания, поэтому все методы по модификации элементов списка кортежу недоступны. В Python кортеж обозначается круглыми скобками. t1 = (1,"4",3.15, 'adc', 7)


**Множество** – структура, которая содержит только неповторяющиеся неупорядоченные элементы. Так как элементы не упорядочены, к ним нельзя обратиться по индексу.
В Python множества обозначается служебным словом set и записывается в фигурных скобках. Множество может быть преобразовано из списка и других типов, что автоматически удалит в исходном объекте дубликаты. Наибольший интерес представляют методы множеств, которые позволяют упростить работы с некоторыми последовательностями. set1={1,4,11,2,4, 8}
- union(other) — возвращает новое множество с элементами из обоих (объединение)
- intersection(other) — возвращает новое множество с общими элементами (пересечение)
- difference(other) — возвращает новое множество с элементами, которых нет в другом
- symmetric_difference(other) — возвращает новое множество с элементами, не общими для обоих


**Словарь** представляет собой структуру с парой «ключ-значения», где ключ - неизменяемая уникальная переменная. В Python словарь обозначается фигурными скобками, а ключ и значение связаны между собой двоеточием. Доступ к элементам словаря осуществляется по ключу, который следует указать в квадратных скобках. d1={'key1':'vv', 'key2':3, 4: 2}
