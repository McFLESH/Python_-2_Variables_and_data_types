**Основы Python. Тема 2. Переменные и типы данных языка Python и работа с ними**


**Краткий конспект**


Процесс создания переменной и присваивания ей значения называется инициализацией переменной. a = 'name'


- Название переменной может содержать только буквы, числа и нижнее подчеркивание;
- Название не может начинаться с числа;
- Название должно быть осмысленным и понятным (например, my_age или user_name);
- Изменяемые в программе переменные всегда называются буквами в нижнем регистре.


Константы пишутся в верхнем регистре. CONST1 = "Python"


**Простые типы данных и их классы**
- Строка - str
- Целое число - int
- Число с плавающей точкой - float
- Логическая переменная - bool


Чтобы из одного типа преобразовать в другой, нужно написать класс(переменная)


**Операции над числовыми типами данных:**
- '+' — сложение
- '-' — вычитание
- '*' — умножение
- / — деление (всегда возвращает float)
- // — целочисленное деление
- % — остаток от деления
- ** — возведение в степень
- abs() — модуль числа
- round() — округление
- pow() — степень
- divmod() — частное и остаток
- type() — определение типа
